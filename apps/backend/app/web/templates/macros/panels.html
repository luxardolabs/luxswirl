{# ═══════════════════════════════════════════════════════════════════════════════ LuxSwirl panel macros — Alpine.js-driven slide-over panels. ═══════════════════════════════════════════════════════════════════════════════ A "panel" is a self-contained slide-over overlay rendered into the global `#panel-container` div in base.html. Each invocation is its own `fixed inset-0 z-50` wrapper with Alpine state (`x-data="{ show: false }"`); `x-init` flips `show` after 10ms so the enter transition plays. Close is triggered three ways: - Escape key (window-level Alpine listener) - Backdrop click - Any element with the `data-close-panel` attribute (close button in the header, Cancel button in the footer) - A backend-triggered `closeSidePanel` window event (HX-Trigger) The close sequence is always: show = false; setTimeout(() => document.getElementById('panel-container').innerHTML = '', 300) which gives the leave transition time to play before the DOM is cleared.
