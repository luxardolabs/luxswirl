Brand Colors

LuxSwirl brand palette based on sky blue (#0ea5e9). Defined in tailwind.config.js as brand-50 through brand-950.

Key Shades (commonly used in templates)

400

Text accent

500

Primary

600

Borders, bg/20

Full scale (50-950) defined in tailwind.config.js. Most UI uses 400 (text), 500 (primary), 600 (borders/backgrounds at /20 opacity).

{# ═══════════════════════════════════════════════════════════════════════ SURFACE COLORS ═══════════════════════════════════════════════════════════════════════ #}

Surface Colors

Dark theme background, text, and border tokens. Each surface has a specific role in the layout hierarchy.

{# Background hierarchy with usage examples #}

Background Hierarchy

{# bg-primary #}

bg-dark-bg-primary #0f172a

Page background, main content area. The darkest surface — everything sits on top of this.

Used by: body, main content area, chart/sparkline inner backgrounds

{# bg-secondary #}

bg-dark-bg-secondary #1e293b

Cards, sidebar, side panels, modals. The primary container surface — one step above the page.

Used by: .card, sidebar, side-panel, table row hover

{# bg-tertiary #}

bg-dark-bg-tertiary #334155

Nested content inside cards — info boxes, key-value grids, code blocks. The lightest surface.

Used by: info cards inside panels, input backgrounds, stat card icons, table headers
