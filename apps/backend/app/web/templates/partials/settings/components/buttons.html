Composite Patterns

Repeated UI primitives beyond simple buttons. From macros/buttons.html.

{# Tag pill button #}

tag_pill_button

Clickable tag in a tag-browser/picker. Subtler than icon_btn (uses /10 bg opacity). Use when the user is selecting from a list of tags.

{{ tag_pill_button('production', 'brand') }} {{ tag_pill_button('staging', 'green') }} {{ tag_pill_button('database', 'blue') }} {{ tag_pill_button('canary', 'yellow') }} {{ tag_pill_button('legacy', 'orange') }} {{ tag_pill_button('deprecated', 'red') }} {{ tag_pill_button('admin', 'purple') }}

Args: label, color (brand/green/blue/yellow/orange/red/purple), extra_classes, extra_attrs.

{# Collapse toggle #}

collapse_toggle

Section/accordion header. Renders a label + rotating chevron. Caller wires the JS hook via toggle_attr.

{{ collapse_toggle('Performance Tuning', toggle_attr='data-demo-toggle-1') }}
{{ collapse_toggle('Reporter Configuration', toggle_attr='data-demo-toggle-2') }}

Args: label, chevron_id, toggle_attr (default data-toggle-collapse), extra_classes, label_classes, extra_attrs. Override extra_classes for non-card layouts (subtle, section, etc).

{# Toggle switch #}

toggle_switch

Pill-style on/off switch. Per action mapping above: active = green. Two sizes — sm for inline/table use, md for prominent settings cards.

{{ toggle_switch(true, size='md', title='Example switch (medium, on)') }} md / on
{{ toggle_switch(false, size='md', title='Example switch (medium, off)') }} md / off
{{ toggle_switch(true, size='sm', title='Example switch (small, on)') }} sm / on
{{ toggle_switch(false, size='sm', title='Example switch (small, off)') }} sm / off

Args: enabled (bool), size (sm|md), on_color (default green; brand available for non-enable semantics), hx_post, hx_target, hx_swap (default outerHTML), hx_trigger, title, extra_attrs.

{# Link button #}

link_btn

Text-only button styled as an inline link. Use for "Manage", "Show details", "Generate" inline actions where a full button would be too heavy.

{{ link_btn('Manage', color='brand') }} {{ link_btn('Show details', color='blue') }} {{ link_btn('Generate Key', color='brand') }} {{ link_btn('Revoke access', color='red') }}

Args: label, color (follows action mapping above), hx_get, hx_post, hx_target, hx_swap, hx_confirm, extra_classes, extra_attrs.
